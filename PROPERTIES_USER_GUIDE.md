# Properties Feature - User Guide

## Quick Start

### Adding a New Property

1. Navigate to **Properties** page
2. Click **"+ Add Property"** button
3. Fill in the required information:
   - **Basic Information**: Title, description, type, bedrooms, bathrooms
   - **Location**: Street address, area, city
   - **Financial Details**: Monthly rent, deposit, service charge
   - **Amenities**: Select applicable amenities
   - **Images**: Upload property photos (up to 10)
4. Click **"Create Property"**

### Viewing Property Details

1. Click on any property card from the list
2. View comprehensive information across 4 tabs:
   - **Overview**: Basic property information
   - **Investment**: Financial analysis and ROI
   - **Contacts**: Related people (tenants, buyers, agents)
   - **History**: Timeline of events

### Filtering Properties

Use the filter bar to find properties:
- **Search**: Type property name, area, or description
- **Status**: Filter by available, occupied, or maintenance
- **Type**: Filter by apartment, house, studio, etc.
- **Area**: Enter specific area (e.g., Kilimani)
- **Rent Range**: Set minimum and maximum rent
- Click **"Clear Filters"** to reset

## Advanced Features

### Investment Analysis

Track and analyze property investment performance:

1. Open property details
2. Go to **"Investment"** tab
3. Click **"+ Add Investment Data"** or **"‚úèÔ∏è Edit Analysis"**
4. Enter:
   - Purchase price and date
   - Renovation costs
   - Current value
   - Projected/actual rental income
   - Monthly expenses (tax, insurance, maintenance, utilities, management)
5. Click **"Calculate Metrics"**

The system automatically calculates:
- **Cap Rate**: Return on investment percentage
- **Monthly Cash Flow**: Net income after expenses
- **ROI**: Overall return on investment

### Linking Contacts

Connect people to properties (tenants, buyers, agents):

1. Open property details
2. Go to **"Contacts"** tab
3. Click **"+ Link Contact"**
4. Search for the contact
5. Select relationship type:
   - Interested
   - Viewed Property
   - Offer Made
   - Current Tenant
   - Owner
   - Agent
6. Add notes (optional)
7. Click **"Link Contact"**

### Managing Images

Upload and organize property photos:

1. When creating/editing a property, scroll to **"Property Images"** section

3. Select up to 10 images

## Tips & Best Practices

### Property Titles
- Be descriptive: "2BR Apartment in Kilimani with Parking"
- Include key features that attract tenants
- Keep it under 100 characters

### Descriptions
- Minimum 50 characters required
- Highlight unique features
- Mention nearby amenities (schools, shopping centers, transport)
- Be honest about the property condition

### Pricing
- Research market rates in your area using the Statistics page
- Set competitive rent based on location and amenities
- Consider including utilities in the rent for simplicity

### Images
- Upload high-quality photos
- Include photos of all rooms
- Show exterior and common areas
- Set the most attractive photo as primary
- Add captions to highlight features

### Amenities
Select all applicable amenities:
- Parking
- Security/CCTV
- WiFi
- Gym/Pool
- Garden/Balcony
- Elevator
- Backup Generator
- Water Tank
- Playground
- Laundry
- Shopping Center

### Investment Tracking
- Update investment data regularly
- Track actual vs projected income
- Monitor expenses monthly
- Use metrics to make informed decisions
- Add notes about major repairs or improvements

## Common Workflows

### Listing a New Property for Rent

1. Add property with complete information
2. Upload attractive photos
3. Set status to "Available"
4. Enable "List Property Publicly"
5. Share property link with potential tenants

### Moving in a Tenant

1. Open property details
2. Go to Contacts tab
3. Link the tenant contact with "Current Tenant" relationship
4. Update property status to "Occupied"
5. Record lease start and end dates

### Property Maintenance

1. Change property status to "Maintenance"
2. Add history entry describing the issue
3. Link vendor contact if applicable
4. Update status back to "Available" when complete

### Analyzing Investment Performance

1. Go to Statistics page to see market averages
2. Open your property's Investment tab
3. Compare your metrics to market averages
4. Adjust rent if needed based on performance
5. Track ROI over time

## Troubleshooting

### Images not uploading?
- Check file size (keep under 5MB per image)
- Ensure images are in JPG, PNG, or WebP format
- Try uploading fewer images at once

### Can't find a property?
- Check your filters - click "Clear Filters"
- Verify you have permission to view the property
- Use the search bar with different keywords

### Investment metrics showing N/A?
- Ensure you've entered purchase price
- Add projected or actual rental income
- Fill in expense information
- Click "Calculate Metrics" button

### Contact not appearing in link modal?
- Verify the contact exists in your Contacts page
- Try searching by email or phone number
- Create the contact first if needed
