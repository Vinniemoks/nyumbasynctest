# MFA Quick Start Guide

## 🚀 5-Minute Setup

### For Admins

#### Step 1: Enable MFA (2 minutes)
1. Login to Admin Dashboard
2. Click **"Security (2FA)"** in sidebar
3. Click **"Enable 2FA"** button
4. Download an authenticator app:
   - Google Authenticator
   - Microsoft Authenticator
   - Authy

#### Step 2: Scan QR Code (1 minute)
1. Open your authenticator app
2. Tap "Add account" or "+"
3. Scan the QR code displayed
4. Or manually enter the secret key

#### Step 3: Verify (1 minute)
1. Enter the 6-digit code from your app
2. Click "Verify & Continue"

#### Step 4: Save Backup Codes (1 minute)
1. Click "Download Backup Codes"
2. Store them securely (password manager recommended)
3. Click "Complete Setup"

✅ **Done! Your account is now protected with 2FA**

---

## 🔐 Daily Login

### With Authenticator App
1. Enter email and password
2. Open authenticator app
3. Enter the 6-digit code
4. Click "Verify"

### With Backup Code (if you lost your phone)
1. Enter email and password
2. Click "Use backup code"
3. Enter one of your backup codes
4. Click "Verify"

---

## 🛠️ Management

### Regenerate Backup Codes
```
Dashboard → Security (2FA) → Regenerate Backup Codes
```

### Disable MFA
```
Dashboard → Security (2FA) → Disable 2FA → Enter code
```

---

## ⚠️ Important Notes

- **Backup codes are single-use** - Each code works only once
- **Save your backup codes** - You won't see them again
- **Keep your phone secure** - Lock it with PIN/biometric
- **Account locks after 5 failed attempts** - Wait 15 minutes

---

## 🆘 Troubleshooting

| Problem | Solution |
|---------|----------|
| Code not working | Check device time is synced |
| Lost authenticator | Use backup code |
| Account locked | Wait 15 minutes |
| QR won't scan | Enter secret key manually |

## 📱 Recommended Apps

**Best Overall**: Authy (multi-device, cloud backup)  
**Most Secure**: Google Authenticator (offline only)  
**Best for Microsoft**: Microsoft Authenticator (push notifications)

---

## 🎯 Security Tips

✅ **DO**
- Store backup codes in password manager
- Use device lock (PIN/fingerprint)
- Keep authenticator app updated
- Review audit logs regularly

❌ **DON'T**
- Share your codes with anyone
- Screenshot QR codes
- Store codes in plain text
- Use same device for everything
